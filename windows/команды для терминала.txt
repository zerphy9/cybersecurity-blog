команды для информации о системе:	
	winver: вызовет окно с информацией об ОС

команды для сортировки вывода:
	| select: позволяет сортировать выданное терминалом значение
		примерный синтаксис: Get-WmiObject -Class win32_OperatingSystem | select Version,BuildNumber
	| format-list: позволяет вывести в удобном виде все значения
	| where-object {$_.<название столбца> -match "значение1|значение2"}: для поиска определенных значений в указанном столбце
		примерный синтаксис: | Where-Object { $_.DisplayName -match "update|upgrade|patch|installer" -or $_.Name -match "update|upgrader" }
		флаги:
			-or $_.<название столбца> -match "значение1|значение2"} - позволяет расширить команду
