распространенные технологии удаленного доступа:
	VPN - Виртуальные частные сети
	SSH - Безопасная оболочка
	FTP - Протокол передачи файлов
	VNC - Виртуальные сетевые вычисления
	WinRM - Удаленное управление Windows
	RDP - Протокол удаленного рабочего стола

RDP - протокол удаленного рабочего стола:
	встроен в системы Windows, по умолчанию прослушивает порт 3389
	подключение Windows - Windows:
		осуществляется встроенной программой RDC (Remote Desktop Connection)
		для работы, необходимо разрешить в системе Windows доступ к удаленному подключению (по умолч. отключен)
	подключение Linux - Windows:
		осуществляется утилитой xfreerdp
		синтаксис: xfreerdp /v:<targetIp> /u:htb-student /p:Password

SMB (Server Message Block) - сетевой протокол прикладного уровня. используется чтобы делиться с файлами
	использует протокол TCP с портом 445
	mount - команда в Linux для подключения файловой системы к общей структуре каталогов
		синтаксис: mount <опции> <устройство или ресурс> <точка монтирования файлов>
		примерный синтаксис: sudo mount -t cifs -o username=admin,password=1234 //192.168.1.5/shared /mnt/share
		опции:
			-o loop: позволяет работать с образом диска как с виртуальным диском
			-t cifs -o username=<username>,password=<password>: для подключения к SMB
				требует установки, имя пакета: cifs-utils
	umount - команда в Linux для отсоединения файловой системы
		синтаксис: umount <точка монтирования файлов>
		примерный синтаксис: sudo umount /mnt/usb
	net share: команда для просмотра всех общих папок в системе
