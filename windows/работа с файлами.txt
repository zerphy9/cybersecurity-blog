команды для информации о файлах:
	dir - выведет содержимое текущей папки - файлы, каталоги, размеры, даты изменения
		примерный синтаксис: dir C:\Windows /p
		ключи:
			/p - выводит постгранично, нажать любую клавишу для продолжения
			/w - широкий режим - только имена в несколько колонок
			/s - показывает файлы и папки во всех подкаталогах
			/b - только имена файлов и папок (без дат и размеров)
			/o:n - сортирует по имени (n - name)
			/t:w - показывает время изменения (w - write)
	tree - позволяет вывести полное дерево папок
	cd - для перехода в папку
		cd .. - назад на уровень вверх
		cd \ - в корень диска
	type <filename> - просмотр содержимого файла

команды для создания файлов:
	mkdir <name> - создать папку
	type nul > <file>.txt - создать пустой текстовый файл

команды для удаления файлов:
	del <filename> - удалить файл
	rmdir <name> - удалить пустую папку
	rmdir /s <name> - удалить папку и его содержимое

копирование и перемещение:
	copy <filename1> <filename2> - копировать файл
	xcopy <name> - копировать папку
	move <filename> <new_fileway> - переместить файл
