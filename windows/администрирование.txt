набор инструментов sysinternals:
	используются для администрирования и диагностики систем Windows
	можно загрузить с веб-сайта Microsoft / использовать напрямую из сайта \\live.sysinternals.com\tools
	включает в себя следующие утилиты:
		Process Explorer - замена диспетчеру задач
		Autoruns - автозагрузка Windows
		PsExec - удаленное выполнение команд и подключение к системам по протоколу SMB
		TcpView - для отслеживания интернет-активности
		ProcMob, Handle...

WMI (Windows Management Instrumentation):
	набор мощных инструментов для регулирования компьютером
	содержит информации для мониторинга системы:
		информации о запущенных процессах, состоянии локальных/удаленных систем
		информация о железе (CPU, диск, BIOS)
		информация о событиях, драйверах, обновлениях
		управление службами, пользователями, сетевыми адаптерами
		настройка параметров безопасности...
	wwic: доступ следующей командой в терминале cmd
		синтаксис: wmic <класс> <операция> <параметры>
		классы: process, cpu, useraccount, bios
		операции: list, call, get
		параметры: name, caption, status, brief (основа)
		команды:
			wmic /? - для получения списка команд и псевдонимов
			wmic os list brief - для получения информации об ОС
	get-wmiobject: для использования с PowerShell
		синтаксис: Get-WmiObject -Class <класс> [-Namespace <пространство>] [-ComputerName <удаленный хост>] [-Filter <условие>]
		примерный синтаксис: Get-WmiObject -Class win32_OperatingSystem
		классы:
			win32_OperatingSystem: для получения информации об установленной системе
			win32_Process: для получения списка процессов
			win32_Service: для получения списка служб
			win32_Bios: для получения информации о биос
			win32_Processor: для получения информации о процессоре
	invoke-wmimethod: для использования метода wmi (завершить процесс, перезагрузить компьютер, запустить службу, вызвать аппаратную перезагрузку, изменить настройки)
		синтаксис: Invoke-WmiMethod -Class <wmi-класс> -Name <метод> [-ArgumentList <аргументы>] [-ComputerName <удаленный хост>]
		методы:
			Terminate - завершить процесс
			Reboot/Shutdown - перезагрузить/выключить систему
			CancelAllJobs - очистить очередь печати

MMC (Microsoft Management Console) - платформа для запуска и создания оснастков-cнапинов (файлов с разрешением .msc)
	.msc - файл-конфигурация для управления определенной части системы (прим. services.msc - управлять службами)
		хранятся в C:\Windows\System32\
	с помощью MMC, оснастки можно добавлять как для локальной, так и удаленной системы

управление пользователями Windows:
	New-LocalUser - команда PowerShell для создания нового пользователя
		синтаксис: New-LocalUser -Name "имя_пользователя" -Password (ConvertTo-SecureString "пароль" -AsPlainText -Force) -FullName "Полное имя" -Description "Описание"
	New-LocalGroup - команда PowerShell для создания группы
		синтаксис: New-LocalGroup -Name "имя_группы"
	Get-LocalGroup - команда PowerShell для получения информации о группе
		синтаксис: Get-LocalGroup -Name "имя_пользователя"
		ключи:
			Get-LocalGroup - получить список доступных групп
			-Name "имя_пользователя" - получить информацию об отдельной группe
	Add-LocalGroupMember - команда для добавления пользователя в группу
		синтаксис: Add-LocalGroupMember -Group "имя_группы" -Member "имя_пользователя"
